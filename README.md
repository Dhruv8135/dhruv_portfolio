# Dhruv Kumar - Full Stack Developer Portfolio

Welcome to my personal developer portfolio! This is a simple, professional, and visually engaging single-page web portfolio built to showcase my skills, projects, and work experience.

## ✨ Features
- **Modern Premium Design:** A clean, minimal dark theme utilizing smooth gradients, "glassmorphism" frosted cards, and subtle CSS animations.
- **Responsive Layout:** fully optimized for both desktop and mobile viewing.
- **Section Parsing:** Includes designated sections for Hero, About Me (Education & Certifications), Experience, Featured Projects, Skills, and Contact Info.

## 🛠️ Tech Stack
- **HTML5:** Clean semantic structure.
- **CSS3:** Custom properties (variables), Flexbox/Grid layouts, custom keyframe animations, responsive media queries.
- **JavaScript:** Minimal vanilla JS for logic such as the intersecting observer scrolling animations and mobile navbar toggles.
- **Icons:** FontAwesome integration.
- **Fonts:** System fonts and `Inter` via Google Fonts.

## 🚀 Getting Started Locally
You can run this project locally with zero dependencies or installations required!
1. Clone this repository to your local machine.
2. Open the project folder.
3. Simply double-click the `index.html` file to run it in your default web browser.

Alternatively, if using VS Code, use the **Live Server** extension for an optimal hot-reloading development experience.

## 🔗 Deployment
This static webpage is easily deployable to platforms like [Vercel](https://vercel.com) or [GitHub Pages](https://pages.github.com/).
